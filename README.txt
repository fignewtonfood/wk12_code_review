# _Shift Cipher_
## _Website for creating a custom shift cipher 10/23/2015_
### By Tim White
## Description
_Users may create account and sign in to access custom cipher module. Cipher modules allows for direction and offset inputs for shift cipher._

## Setup

Clone https://github.com/fignewtonfood/wk12-code-review.git

_This project makes use of: Apache, MySql, PHP, Drupal with custom Shift Cipher module._
